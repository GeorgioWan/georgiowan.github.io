[5月] 街口的電影院 - 月總覽

發表於 2016-05-31   |   分類於 Movie   |     |  
  |  


　　5 部韓國電影，2 部好萊塢電影，2 部中國電影，1 部台灣電影，1 部印度電影，共 11 部，這個五月看了不少韓國電影呢！下面，就讓我們用 比較不雷 的方式，分享這個月看了哪些電影吧！

盲證 (Blind) 2011

  • 驚悚 / 懸疑
  • 主要演員：金荷娜《樂園》、俞承浩《有你真好》
  • IMDb 6.8 │ 爛番茄 暫無 │ OA

　　整部片營造的氛圍，如同盲眼的女主角一樣，就是「不安」！這部片充分讓人感受到那種背脊發涼的感覺。

　　除了不安氛圍營造得很好，還有演技超群的女主角 金荷娜 外，讓我感到驚喜的就是長大後還是演小屁孩的 俞承浩！來幫大家回味一下經典好片《有你真好》中可愛又欠揍的小屁孩。

　　但在盲證裡，他已經是帥帥的小鮮肉了！可能是只看過他這兩部作品，所以真心覺得他是演小屁孩界的翹楚 (蓋章) XDD

　　整體上這部片是不錯的，如果會害怕，推薦你選個風和日麗的下午茶時間欣賞，哈哈哈。

檢察官外傳 (A Violent Prosecutor) 2016

  • 喜劇 / 犯罪
  • 主要演員：姜棟元、黃正民
  • IMDb 5.9 │ 爛番茄 暫無 │ OA

　　雖然劇情稍嫌平淡，但不得不說 黃正民 和 姜棟元 的演技生動的帶領整部電影，他們在這各自角色的設定上完整發揮了！

　　除了這兩位主角外，配角 李聖旻 也讓我很驚喜，就是《未生》中的吳相植次長，看他演戲真的很有趣。

　　這部片雖然只給了兩顆半的推薦，但還是很適合閒暇時，配個晚餐輕鬆一下的電影。

地獄奶奶 (Granny’s Got Talent) 2015

  • 喜劇 / 家庭
  • 主要演員：金秀美《奶奶強盜團》、李泰蘭《家有七公主》、鄭萬植
  • IMDb 暫無 │ 爛番茄 暫無 │ OA

　　說實在我不太記得詳細的劇情發生了什麼事，可能是劇情安排過於老派，也可能是對於「韓國髒話」的理解不是那麼深，所以也比較難融入，而感情的部份也覺得沒到點上 …。不過，意外的在劇中看到應該穿著軍服的 羅雪七(李泰蘭) XDDD
　　總而言之，不太推薦。

我愛你 (I Love You) 2011 大推薦

  • 家庭 / 愛情
  • 主要演員：李順載《摘星星給我》、金秀美《奶奶強盜團》、宋在浩《相信愛情》、宋智孝《Running Man》、《霜花店》
  • IMDb 暫無 │ 爛番茄 暫無 │ OA

　　沒有轟轟烈烈的激盪、也沒有過度煽情的話語，這個故事裡，只有 4 個老人，2 份深刻的感情，執手相老。
　　很開心在 小凱蘿 的極力推薦下看了這部作品，這部作品中有太多讓人掉眼淚的地方，請記得準備多點衛生紙 (擦)。

　　在銀髮族的愛情世界裡，很多事情都變得不那麼複雜，但其實愛情本身，何嘗不是如此呢？

　　除了深刻的描寫 愛情 外，家庭 和 友情 也恰到好處的點亮整部作品。

　　當然這不單單只是讓人哭哭啼啼的電影，在人物與人物間的設定上，巧妙的加入一些詼諧當點綴，不多不少，剛剛好！

　　現在，換我極力推薦這部作品給大家，千萬別因為是老人家當男女主角就先拒於門外唷！

窒命地 (Colonia) 2015

  • 劇情 / 愛情 / 驚悚 / 歷史
  • 主要演員：艾瑪華森《哈利波特》、丹尼爾布爾《決戰終點線》
  • IMDb 7.2 │ 爛番茄 23% │ OA

　　有些朋友聽到電影名稱或看到電影海報，都會誤以為是一部 鬼片，但其實這是一部真人真事改編的 驚悚片！對於場景和人物 (除了男女主角) 的還原度沒有 100% 也已經 99.9% 了，看到片尾的真實照片，整個汗毛豎到頭頂了都。

　　劇中最大的賣點之一就是造型多端的 艾瑪華森 和她精湛的演技，到底怎麼可以讓人這麼漂亮、這麼會唸書又把自己管理得這麼好(怒)。

　　這部片所描述的事件，背景是在 1973 年的智利政變，一個德式小農莊 (Colonia Dignidad) 背後所掩藏的駭人秘密，可參考 尊嚴殖民地：智利歷史上最黑暗的一頁。

　　其實還蠻推薦去看的，不過，可能是真人真事改編，看完心有餘悸了一兩天呢…！

北京遇上西雅圖 (Finding Mr. Right) 2013

  • 愛情
  • 主要演員：吳秀波《黎明之前》、湯唯《色，戒》
  • IMDb 6.4 │ 爛番茄 67% │ OA

　　我是在看完《不二情書》的隔天才立馬追了這部作品，雖然都是片名和演員都是一樣的，但故事內容 完全不同 且不馬虎！

　　這部作品很有趣，北京和西雅圖完全刻化在男女主角的身上，保守、傳統、固執卻紳士，遇上　拜金、膚淺、無賴卻正直，雖然算是蠻典型的浪漫故事，但劇本編寫得很流暢很舒服。

　　一開始除了對湯唯有點印象外，對吳秀波一點都不認識，但這兩部片後讓我覺得他根本就是亞洲小勞勃，帥屎了！

北京遇上西雅圖之不二情書 (Book of Love) 2016

  • 愛情 / 劇情
  • 主要演員：吳秀波《黎明之前》、湯唯《色，戒》
  • IMDb 6.9 │ 爛番茄 暫無 │ OA

　　不得不說，這是我很喜歡的一部作品！雖然直直白白的把電影內容打在標題上，但卻能給出超值的感動！這次的故事背景就不再是北京和西雅圖，而是 澳門遇上舊金山 XDD

　　我想，或多或少許是為了宣傳澳門賭場和促進澳門觀光，在這部作品的場景中除了賭場外，吸引我的還有那些街道巷弄，雖然壅塞但卻很美；有發現劇照中的湯唯，又更美了嗎！(這是什麼巫術)

　　現代人的步調越來越快，通訊已是一兩秒的事，雖然方便，但，我們也因此失去等待、揣測和深層交流的感動。

　　很推薦撥一點時間欣賞這部作品，讓自己沉澱一下，慢下來，細嚼生活中的一言一語，為自己的浪漫找一個伴。
　　
　　有些人，遇見就好。

按讚情人 (Like for likes) 2016

  • 喜劇 / 愛情
  • 主要演員：姜河那《未生》、《二十》、金柱赫《請回答1988》、李絮《布拉芙夫人》、崔智友《天國的階梯》、李美妍《請回答 1988》、劉亞仁
  • IMDb 7.1 │ 爛番茄 暫無 │ OA

　　劇情整體來說，故事得進行很流暢，但比較平淡簡單。

　　劇中 6 對男女的角色設定都很可愛，有一種說不出來的特質，可能就是單純的 可愛 吧！

　　這部小品還是很適合選一個週末悠閒的品嚐，很輕鬆很愉快，也帶給人情竇初開得幸福感！

我們全家不太熟 (We Are Family) 2015 請三思

  • 喜劇 / 愛情 / 家庭
  • 主要演員：張榕容《逆光飛翔》、《渺渺》、張書豪《女朋友，男朋友》、郝劭文《那些年，我們一起追的女孩》、陳松勇
  • IMDb 暫無 │ 爛番茄 暫無 │ OA

　　在這部作品裡，我看到很會演戲的實力派演員 張榕容、陳松勇 和復出後演戲越來越自然的兒時最愛 郝劭文，但只能說好的演員儘管在努力，也不敵劇本和角色設定上的缺陷！或許是我滿心期待的挑了部台片來看，開頭的動畫也相當可愛，讓我把期待拉得太高了，所以才會如此失望吧(？) 雖然給了超低分，但也很高興至少台片的製作有越來越優質取向！

　　雖然不是很推薦看，但裡面有可愛的 豆寶 和 張榕容 唷！

刺激1995 (The Shawshank Redemption) 1994 必看

  • 劇情 / 犯罪
  • 主要演員：摩根費里曼《王牌天神》、Tim Robbins《神秘河流》
  • IMDb 9.3 │ 爛番茄 98% │ OA

　　因為很喜歡看電影，平時除了挖掘一些冷門片外，也會找經典必看的作品來看看，這次的經典好片真的不讓人失望！還沒看過的或正想看的人不要再猶豫了，必看！無論是演員的演技、劇情的編寫和節奏還是畫面的美感，都讓人感到舒服且記憶深刻。

　　「眨眼間，整個生活都被毀了。你一無所有，除了餘生的時間來思考。大多數新來的人第一晚都幾近發瘋，總有人會哭出來，每次都是。問題是，會是誰呢？」

　　「監獄裡的高牆實在是很有趣。剛入獄的時候，你痛恨周圍的高牆；慢慢地，你習慣了生活在其中；最終你會發現自己不得不依靠它而生存，這就是體制化。」

　　「生命可以歸結為一種簡單的選擇：要麼忙於生存，要麼趕著去死。」

　　“It takes a strong man to save himself，and a great man to save another.”
　　
　　整部作品的起承轉合都很完美，推薦推薦！！

對不起我愛你 (Sanam Teri Kasam) 2016 請三三三思

  • 愛情
  • 主要演員： Harshvardhan Rane、Mawra Hocane
  • IMDb 7.5 │ 爛番茄 98% │ OA

　　怎麼說呢…除了男主角 身材超好，女主角 長的美美der 外，無論是劇情、配樂還是演技方面，我真的有看沒有懂︶︵︶
　　就讓我們放上一張猛男照，結束這回合。

結語

　　終於打完了！透過撰寫點評心得的方式來回味這個月看的電影還蠻值得的，除了能夠品嚐更多的細節外，還能夠為自己留下點什麼！但是，真的好累RRRRR！
